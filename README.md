# みんなでしりとり！ ワードバスケット

[![Deploy to GitHub Pages](https://github.com/GIGAyama/word_basket/actions/workflows/deploy.yml/badge.svg)](https://github.com/GIGAyama/word_basket/actions/workflows/deploy.yml)

## 概要
「ワードバスケット」は、みんなでわいわい楽しめるリアルタイムしりとりカードゲームのWebアプリ版です。
小学生から大人まで、直感的な操作で遊べるようにデザインされています。

**公開URL:** [https://gigayama.github.io/word_basket/](https://gigayama.github.io/word_basket/)

## 特徴
- **直感的なUI/UX**: 小学生でも迷わず遊べる、大きなボタンとわかりやすいアイコンを採用。
- **マルチプレイヤー対応**: 2人から最大6人まで、端末1つを囲んで（または画面共有で）遊べます。
- **カスタマイズ可能なタイマー**: 10秒の激ムズモードから無制限まで設定可能。
- **サウンド演出**: ゲームを盛り上げる効果音とBGM（ON/OFF切替可能）。
- **特殊カード対応**: 「あ行」ワイルドカードや「5文字以上」などの特殊ルールを完全再現。

## 遊び方
1. **設定**: 参加人数と制限時間を選んでスタート！
2. **しりとり**: 「場のカード」の文字で始まり、「自分の手札」の文字で終わる3文字以上の言葉を言います。
3. **カードを出す**: 言葉を言ったら、そのカードをタップして場に出します。
4. **勝利条件**: いち早く手札をすべてなくしたプレイヤーの勝利です！

## 技術スタック
- **Vite**: 高速なビルドツール
- **React**: UIコンポーネントライブラリ
- **Tailwind CSS**: レスポンシブなスタイリング
- **Lucide React**: 美しいアイコンセット
- **GitHub Actions**: 自動デプロイ環境 (CI/CD)
